# Flashcard Generator

A web application for creating, managing, and studying digital flashcards with advanced features like AI-powered card generation, progress tracking, and interactive quiz modes.

## Features

- Create custom flashcards with questions and answers
- Organize cards by difficulty levels (Easy, Medium, Hard)
- Interactive quiz mode with timer options
- Progress tracking and performance analytics
- Dark/Light theme support
- Import/Export functionality
- Local storage for data persistence
- Search and filter capabilities
- Star important cards
- Identify weak areas for improvement

## Technologies Used

- **HTML5** - Semantic structure and modern web standards
- **CSS3** - Responsive design with Glassmorphism effects
- **JavaScript (ES6+)** - Modern JavaScript with classes and modules
- **Local Storage API** - Client-side data persistence

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Start creating and studying flashcards!

## Usage

1. **Create Flashcards**: Add questions and answers with difficulty levels
2. **Study Mode**: Take interactive quizzes to test your knowledge
3. **Track Progress**: Monitor your learning progress and identify weak areas
4. **Import/Export**: Save your flashcards and share with others
